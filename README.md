Customer Retention & Churn Insights Dashboard
🎯 Project Objective
This dashboard identifies customer attrition patterns to help businesses reduce churn and protect revenue. It connects a MySQL database to Power BI to provide real-time strategic insights.

🛠️ Tech Stack
Data Source: MySQL (churn_analysis database)

Visualization: Power BI Desktop

Data Processing: Power Query (ETL)

Analysis: DAX (Data Analysis Expressions)

📊 Key Insights
Financial Impact: Visualized revenue through TotalCharges analysis.

Loyalty Trends: Tracked tenure to identify high-risk periods for customer loss.

Behavioral Analysis: Correlated PaymentMethod with retention rates.
